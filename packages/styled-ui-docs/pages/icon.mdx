# Icon

`Icon` is used to render SVG icons in a more convenient way. You can extend icons by defining icon keys in `theme.icons`.

## Import

```js
import Icon from '@trendmicro/react-styled-ui/Icon';
// or
import { Icon } from '@trendmicro/react-styled-ui';
```

## Usage

Use an icon by passing the `name` prop. This name must match an icon key defined in `theme.icons`. By default, the icon inherits the font size and color of its parent.

## Adding custom icons

First, you have to add custom icons to the theme. Each icon must be an object containing `path` and optional style props passed to [SVGIcon](./svgicon).

```jsx disabled noInline
const customIcons = {
  icon1: {
    path: (
      <path fill="currentColor" d="..." />
    ),
  },
  icon2: {
    path: (
      <g fill="currentColor">
        <path d="..." />
      </g>
    ),

    // optional style props
    viewBox: '0 0 48 48',
  },
};

const customTheme = {
  ...theme,
  icons: {
    ...theme.icons,
    ...customIcons,
  },
};
```

After that simply wrap your app into `ThemeProvider` component and pass your theme as a `theme` prop. Just like this:

```jsx disabled noInline
<ThemeProvider theme={customTheme}>
  <App />
</ThemeProvider>
```

Pass the icon name as a prop to the `<Icon>` component to render the SVG icon:

```jsx disabled noInline
<Icon name="icon1" color="blue:50" />
```

## Props

`Icon` composes the [`SVGIcon`](./svgicon) component. You can override the default styles with style props.

| Name | Type | Default | Description |
| :--- | :--- | :------ | :---------- |
| name | string | | The name of the icon. |
