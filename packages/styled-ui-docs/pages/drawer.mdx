# Drawer

## Import

Trend Micro Styled UI exports 7 components to help you create any Drawer.

- `Drawer`: The wrapper that provides context for its children
- `DrawerContent`: The content of the Drawer.
- `DrawerHeader`: The header that labels the Drawer
- `DrawerFooter`: The footer that houses the Drawer actions
- `DrawerBody`: The wrapper that houses the Drawer's main content
- `DrawerOverlay`: The overlay of the Drawer.
- `DrawerCloseButton`: The button that closes the Drawer.

```js
import {
  Drawer,
  DrawerContent,
  DrawerOverlay,
  DrawerHeader,
  DrawerFooter,
  DrawerBody,
  DrawerCloseButton,
} from '@trendmicro/react-styled-ui';
```

## Usage

```jsx
function BasicUsage() {
  const { isOpen, onOpen, onClose } = useDisclosure();
  return (
    <>
      <Button onClick={onOpen}>Open Drawer</Button>

      <Drawer isOpen={isOpen} onClose={onClose}>
        <DrawerOverlay />
        <DrawerContent>
          <DrawerHeader title="Drawer Title">Drawer Title</DrawerHeader>
          <DrawerBody>
            <Lorem count={5} />
          </DrawerBody>
          <DrawerFooter>
            <Button mr="2x" variant="primary">Submit</Button>
            <Button onClick={onClose}>
              Cancel
            </Button>
          </DrawerFooter>
        </DrawerContent>
      </Drawer>
    </>
  );
}
```

### Sizes

Pass the `size` prop if you need to adjust the size of the Drawer. Values can be
 `sm`, `md`, `lg`, `full` or `auto`.

```jsx
function SizeExample() {
  const { isOpen, onOpen, onClose } = useDisclosure();
  const [size, setSize] = React.useState('md');

  const handleSizeClick = newSize => {
    setSize(newSize);
    onOpen();
  };

  const sizes = ['sm', 'md', 'lg', 'full', 'auto'];

  return (
    <>
      {sizes.map(size => (
        <Button
          onClick={() => handleSizeClick(size)}
          key={size}
          m={4}
        >{`Open ${size} Drawer`}</Button>
      ))}

      <Drawer onClose={onClose} size={size} isOpen={isOpen}>
        <DrawerOverlay />
        <DrawerContent>
          <DrawerHeader>Drawer Title</DrawerHeader>
          <DrawerBody>
            Bla bla bla...
            Bla bla bla...
            Bla bla bla...
          </DrawerBody>
          <DrawerFooter>
            <Button onClick={onClose}>Close</Button>
          </DrawerFooter>
        </DrawerContent>
      </Drawer>
    </>
  );
}
```

### Placement

Pass the `placement` prop if you need to adjust the placement of the Drawer. Values can be
 `right` or `left`. Default is `right`.

```jsx
function SizeExample() {
  const { isOpen, onOpen, onClose } = useDisclosure();
  const [placement, setPlacement] = React.useState('right');

  const handlePlacementClick = newPlacement => {
    setPlacement(newPlacement);
    onOpen();
  };

  const placements = ['left', 'right'];

  return (
    <>
      {placements.map(placement => (
        <Button
          onClick={() => handlePlacementClick(placement)}
          key={placement}
          m={4}
        >{`Open ${placement} Drawer`}</Button>
      ))}

      <Drawer onClose={onClose} placement={placement} isOpen={isOpen}>
        <DrawerOverlay />
        <DrawerContent>
          <DrawerHeader>Drawer Title</DrawerHeader>
          <DrawerBody>
            Bla bla bla...
            Bla bla bla...
            Bla bla bla...
          </DrawerBody>
          <DrawerFooter>
            <Button onClick={onClose}>Close</Button>
          </DrawerFooter>
        </DrawerContent>
      </Drawer>
    </>
  );
}
```

## Props

### Drawer Props

| Name               | Type              | Default                  | Description |
| :----------------- | :---------------- | :----------------------- | :--------------------- |
| isOpen             | boolean           |                          | If `true`, the drawer is shown |
| placement          |`'left'`, `'right'`| `'right'`                | The drawer placement |
| size               | `'sm'`, `'md'`, `'lg'`, `'full'`, `'auto'`   |  `'md'` | The size of the drawer |
| disableOverlay     | boolean           | `false`                  | If `true`, no shown the overlay when drawer open |
| closeOnEsc         | boolean           | `false`                  | If `true`, close the drawer when the `esc` key is pressed |
| closeOnOverlayClick | boolean          | `false`                  | If `true`, close the drawer when click outside of the drawer |
| onOpen             | () => void        |                          | Callback fired when the drawer opens |
| onClose            | () => void        |                          | Callback fired when the drawer closes |
| initialFocusRef    | React.ref         |                          | The `ref` of the element to receive focus when the dialog opens |
| finalFocusRef      | React.ref         |                          | The `ref` of element to receive focus when the dialog closes |
| autoFocus          | boolean           | `false`                  | If `true` and `initialFocusRef` not set a ref, auto focus on first focusable element of model content |

### Other Props
- `DrawerHeader`, `DrawerFooter` and `DrawerBody` composes [`Box`](./box).
- `DrawerCloseButton` based on [`ButtonBase`](./buttonbase)
