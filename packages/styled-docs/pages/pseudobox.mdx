# PesudoBox

## Import

```js
import PseudoBox from '@trendmicro/react-styled-core/PseudoBox';
// or
import { PseudoBox } from '@trendmicro/react-styled-core';
```

## Usage

```jsx
<PseudoBox
  px={24}
  py={12}
  backgroundColor="#f0f0f0"
  width={200}
  _hover={{
    color: 'whiteAlpha.primary',
    backgroundColor: 'blackAlpha.primary',
  }}
>
  PseudoBox
</PseudoBox>
```
