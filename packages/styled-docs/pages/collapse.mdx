# Collapse

The Collapse component is used to create regions of content that can
expand/collapse with a simple animation. It helps to hide content that's not
immediately relevant to the user.

This component leverages
[react-animate-height](https://github.com/Stanko/react-animate-height), and
composes the [Box](./box) component.

## Import

```js
import Collapse from '@trendmicro/react-styled-core/Collapse';
// or
import { Collapse } from '@trendmicro/react-styled-core';
```

## Usage

### Basic usage

```jsx
function Example() {
  const [show, setShow] = React.useState(false);

  const handleToggle = () => setShow(!show);

  return (
    <div style={{ whiteSpace: 'pre-wrap' }}>
      <Button variantColor="emphasis" onClick={handleToggle}>
        Toggle
      </Button>
      <Collapse mt={4} isOpen={show}>
        Hello, we are Trend Micro Frontend, PR please when had some advice ^_^
      </Collapse>
    </div>
  );
}
```

### Changing the `startingHeight`

```jsx
function Example() {
  const [show, setShow] = React.useState(false);

  const handleToggle = () => setShow(!show);

  return (
    <div style={{ whiteSpace: 'pre-wrap' }}>
      <Collapse startingHeight={20} isOpen={show}>
        Hello, we are Trend Micro Frontend, PR please when had some advice ^_^.
        嗨, 我們是趨勢科技的前端團隊, 歡迎發PR給予我們幫助跟建議。
        謝謝大家賞臉
      </Collapse>
      <Button size="sm" onClick={handleToggle} mt="1rem">
        Show {show ? "Less" : "More"}
      </Button>
    </div>
  );
}
```

## Props

This component doesn't have any custom props.

| Name             | Type      | Description                                                                                                  |
| ---------------- | --------- | ------------------------------------------------------------------------------------------------------------ |
| isOpen           | `boolean` | If `true`, the content will be visible                                                                       |
| animateOpacity   | `boolean` | If `true`, the opacity of the content will be animated                                                       |
| duration         | `number`  | The animation duration as it expands                                                                         |
| startingHeight   | `number`  | The height you want the content in it's collapsed state. Set to `0` by default                               |
| endingHeight     | `number`  | The height you want the content in it's expanded state. Set to `auto` by default                             |
| onAnimationEnd   | `func`    | The function to be called when the collapse animation starts. It provides the `currentHeight` as an argument |
| onAnimationStart | `func`    | The function to be called when the collapse animation ends. It provides the `currentHeight` as an argument   |
