# ButtonGroup

`ButtonGroup` is used to group related buttons.

## Import

```js
import ButtonGroup from '@trendmicro/react-styled-core/ButtonGroup';
// or
import { ButtonGroup } from '@trendmicro/react-styled-core';
```

## Usage

### Basic button group

```jsx
<ButtonGroup>
  <Button>One</Button>
  <Button>Two</Button>
  <Button>Three</Button>
</ButtonGroup>
```

### Group variants

Use the `variant` prop to change the visual style of every button in a group. You can set the value to `emphasis`, `primary`, `default`, `secondary` and `ghost`.

```jsx
<Stack direction="column" spacing="3x">
  <ButtonGroup variant="default">
    <Button>One</Button>
    <Button>Two</Button>
    <Button>Three</Button>
  </ButtonGroup>
  <ButtonGroup variant="secondary">
    <Button>One</Button>
    <Button>Two</Button>
    <Button>Three</Button>
  </ButtonGroup>
</Stack>
```

### Group sizes

Use the `size` prop to change the size of the `ButtonGroup`. You can set the value to `sm`, `md`, or `lg`.

```jsx
<Stack direction="column" spacing="3x">
  <ButtonGroup size="sm">
    <Button>Left</Button>
    <Button>Middle</Button>
    <Button>Right</Button>
  </ButtonGroup>
  <ButtonGroup size="md">
    <Button>Left</Button>
    <Button>Middle</Button>
    <Button>Right</Button>
  </ButtonGroup>
  <ButtonGroup size="lg">
    <Button>Left</Button>
    <Button>Middle</Button>
    <Button>Right</Button>
  </ButtonGroup>
</Stack>
```

### Group orientation

Make a set of buttons appear vertically stacked rather than horizontally, by adding `vertical` to the `ButtonGroup` component.

```jsx
<Stack direction="column" spacing="1rem" shouldWrapChildren>
  <ButtonGroup>
    <Button>One</Button>
    <Button>Two</Button>
    <Button>Three</Button>
  </ButtonGroup>
  <ButtonGroup vertical>
    <Button>One</Button>
    <Button>Two</Button>
    <Button>Three</Button>
  </ButtonGroup>
</Stack>
```

### Button states

```jsx
<Stack direction="column" spacing="3x">
  <Stack direction="row" spacing="3x">
    <ButtonGroup>
      <Button><TMIcon name="chart-pie" /></Button>
      <Button selected><TMIcon name="chart-line" /></Button>
      <Button disabled><TMIcon name="chart-bar" /></Button>
    </ButtonGroup>
    <ButtonGroup>
      <Button>Normal</Button>
      <Button selected>Selected</Button>
      <Button disabled>Disabled</Button>
    </ButtonGroup>
  </Stack>
  <Stack direction="row" spacing="3x">
    <ButtonGroup variant="secondary">
      <Button><TMIcon name="chart-pie" /></Button>
      <Button selected><TMIcon name="chart-line" /></Button>
      <Button disabled><TMIcon name="chart-bar" /></Button>
    </ButtonGroup>
    <ButtonGroup variant="secondary">
      <Button>Normal</Button>
      <Button selected>Selected</Button>
      <Button disabled>Disabled</Button>
    </ButtonGroup>
  </Stack>
</Stack>
```

## Props

`ButtonGroup` composes the [`Box`](./box) component. You can override the default styles with style props.

| Name | Type | Default | Description |
| :--- | :--- | :------ | :---------- |
| size | string | 'md' | The size for all buttons in the group. One of: `'sm'`, `'md'`, `'lg'` |
| variant | string | 'default' | The variant of all buttons in the group. One of: `'emphasis'`, `'primary'`, `'default'`, `'secondary'`, `'ghost'` |
| vertical | boolean | false | Make the set of Buttons appear vertically stacked. |
