# Button

`Button` is used to trigger an action or event, such as submitting a form, opening a [`Modal`](./modal), canceling an action, or performing a delete operation.

## Import

```js
import Button from '@trendmicro/react-styled-core/Button';
// or
import { Button } from '@trendmicro/react-styled-core';
```

## Usage

```jsx
<Button>Default Button</Button>
```

### Variants

Use the `variant` prop to change the visual style of the Button. You can set the value to `solid`, `outline`. Use `outline` to remove all background images and colors on any button.

```jsx
<Stack direction="row" spacing=".5rem" shouldWrapChildren>
  <Button variant="solid">Solid Button</Button>
  <Button variant="outline">Outlined Button</Button>
</Stack>
```

### Colors

Use the `variantColor` prop to change the color scheme of the Button. `variantColor` can be any color key with key `50`(hover), `60`(normal) and `70`(active) that exist in the them.color.

```jsx
<Stack direction="row" spacing=".5rem" shouldWrapChildren>
  <Button variantColor="red">Emphasis Button</Button>
  <Button variantColor="blue">Primary Button</Button>
  <Button variantColor="gray">Default Button</Button>
</Stack>
```

### Sizes

Use the `size` prop to change the size of the button. You can set the value to `sm`, `md`, or `lg`.

```jsx
<Stack direction="row" spacing=".5rem" alignItems="center" shouldWrapChildren>
  <Button size="sm">Small</Button>
  <Button size="md">Medium</Button>
  <Button size="lg">Large</Button>
</Stack>
```

### States

```jsx
<Stack direction="column" spacing="1rem">
  <Stack direction="row" spacing=".5rem" alignItems="center" shouldWrapChildren>
    <Button active>Active</Button>
    <Button>Normal</Button>
    <Button disabled>Disabled</Button>
  </Stack>
  <Stack direction="row" spacing=".5rem" alignItems="center" shouldWrapChildren>
    <Button variant="outline" active>Active</Button>
    <Button variant="outline">Normal</Button>
    <Button variant="outline" disabled>Disabled</Button>
  </Stack>
</Stack>
```

### Icon buttons

Icon buttons are commonly used in toolbars. Sometimes you might want to add label to certain icon buttons to enhance the UX. You can use [Space](./space) to add a padding space with `.5rem` (`8px`) between icon and label.

```jsx
<Stack direction="column" spacing="1rem">
  <Stack direction="row" spacing=".5rem">
    <Button variantColor="blue" borderRadius="circle" width="2rem" height="2rem">
      <TMIcon name="add" color="white" />
    </Button>
    <Button variantColor="blue" width="2rem" height="2rem">
      <TMIcon name="add" color="white" />
    </Button>
    <Button variantColor="blue" borderRadius="2rem">
      <TMIcon name="add" color="white"/>
      <Space width=".5rem" />
      Add Account
    </Button>
    <Button variantColor="blue">
      <TMIcon name="add" color="white"/>
      <Space width=".5rem" />
      Add Account
    </Button>
  </Stack>
  <Stack direction="row" spacing=".5rem">
    <Button variant="outline" borderRadius="circle" width="2rem" height="2rem">
      <TMIcon name="add" />
    </Button>
    <Button variant="outline" width="2rem" height="2rem">
      <TMIcon name="add" />
    </Button>
    <Button variant="outline" borderRadius="2rem">
      <TMIcon name="add" />
      <Space width=".5rem" />
      Add Account
    </Button>
    <Button variant="outline">
      <TMIcon name="add" />
      <Space width=".5rem" />
      Add Account
    </Button>
  </Stack>
</Stack>
```

## Custom button

In case you need to customize the button, pass style props to the [ButtonBase](./buttonbase) component to create a custom button.

```jsx
<ButtonBase
  bg="#f5f6f7"
  border="1px solid"
  borderColor="#ccd0d5"
  borderRadius="md"
  color="#4b4f56"
  fontSize="14px"
  fontWeight="semibold"
  height="24px"
  lineHeight="1.2"
  px="8px"
  transition="all 0.2s cubic-bezier(.08,.52,.52,1)"
  _hover={{
    bg: "#ebedf0"
  }}
  _active={{
    bg: "#dddfe2",
    transform: "scale(0.98)",
    borderColor: "#bec3c9",
  }}
  _focus={{
    boxShadow:
      "0 0 1px 2px rgba(88, 144, 255, .75), 0 1px 1px rgba(0, 0, 0, .15)",
  }}
>
  Join Group
</ButtonBase>
```

## Props

`Button` composes the [`ButtonBase`](./buttonbase) component. You can override the default styles with style props.

| Name | Type | Default | Description |
| :--- | :--- | :------ | :---------- |
| disabled | `boolean` | false | If `true`, the button will be disabled. |
| size | `string` | `md` | The size of the button. One of: 'sm', 'md', 'lg' |
| variant | `string` | 'solid' | The variant of the button style to use. One of: 'solid', 'outline' |
| variantColor | `string` | | The color of the button. It must be a color key defined in `theme.colors`. |
