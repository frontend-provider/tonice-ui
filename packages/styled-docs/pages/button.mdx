# Button

The Button component is used to trigger an action or event, such as submitting a
form, opening a [Dialog](./modal), canceling an action, or performing a delete
operation.

## Import

```js
import Button from '@trendmicro/react-styled-core/Button';
// or
import { Button } from '@trendmicro/react-styled-core';
```

## Usage

```jsx
<Button>Default Button</Button>
```

### Button Sizes

Use the `size` prop to change the size of the button. You can set the value to
`sm`, `md`, or `lg`.

```jsx
<div>
  <Button size="sm">Small Button</Button>
  <Button size="md">Medium Button</Button>
  <Button size="lg">Large Button</Button>
</div>
```

### Button Color

Use the `variantColor` prop to change the color scheme of the Button. You can set the
value to `emphasis`, `primary`, `default`, `secondary`, `ghost`.

```jsx
<div>
  <Button variantColor="emphasis">Emphasis Button</Button>
  <Button variantColor="primary">Primary Button</Button>
  <Button variantColor="default">Default Button</Button>
  <Button variantColor="secondary">Secondary Button</Button>
  <Button variantColor="ghost">Ghost Button</Button>
</div>
```

### Button Variant

Use the `variant` prop to change the visual style of the Button. You can set the
value to `round`, `solid`.

```jsx
<div>
  <Button variant="solid">Solid Button</Button>
  <Button variant="round">Round Button</Button>
</div>
```

## Accessibility

- Button has `role` button.
- When Button has focus, <kbd>Space</kbd> and <kbd>Enter</kbd> activates it.

## Composition

Button composes [PseudoBox](./pseudobox) and all props you pass (variant,
variantColor, etc.) are converted to style props. This means you can override
any style of the Button via props.

```jsx
// The size prop affects the height of the button
// but I can still override it by passing a custom height
<Button
  size="md"
  height="48px"
  width="200px"
  border="2px solid"
  borderColor="green.50"
>
  Button
</Button>
```

## Custom Button

In event you need to make your own custom button, you can leverage the
`PseudoBox` component. It provides the `hover`, `focus`, `active` and `disabled`
style props to style the button.

```jsx
// Button from facebook.com
<PseudoBox
  as="button"
  height="24px"
  lineHeight="1.2"
  transition="all 0.2s cubic-bezier(.08,.52,.52,1)"
  border="1px solid"
  px="8px"
  fontSize="14px"
  fontWeight="semibold"
  bg="#f5f6f7"
  borderColor="#ccd0d5"
  color="#4b4f56"
  _hover={{ bg: "#ebedf0" }}
  _active={{
    bg: "#dddfe2",
    transform: "scale(0.98)",
    borderColor: "#bec3c9",
  }}
  _focus={{
    boxShadow:
      "0 0 1px 2px rgba(88, 144, 255, .75), 0 1px 1px rgba(0, 0, 0, .15)",
  }}
>
  Join Group
</PseudoBox>
```

## Props

The Button composes the `PseudoBox` component so you can pass props for
`PseudoBox`. These are props related to the Button component.

| Name           | Type                                                     | Default   | Description                                                      |
| -------------- | -------------------------------------------------------- | --------- | ---------------------------------------------------------------- |
| `variant`      | `round`, `solid`                                         | `solid`   | The variant of the button style to use.                          |
| `variantColor` | `emphasis`, `primary`, `default`, `secondary`, `ghost`   | `default` | The color scheme of the button.                            |
| `disabled`     | `boolean`                                                |           | If `true`, the button will be disabled.                          |
| `size`         | `sm`, `md`, `lg`                                         | `md`      | The size of the button.                                          |
