# Button

The Button component is used to trigger an action or event, such as submitting a form, opening a [Dialog](./modal), canceling an action, or performing a delete operation.

## Import

```js
import Button from '@trendmicro/react-styled-core/Button';
// or
import { Button } from '@trendmicro/react-styled-core';
```

## Usage

```jsx
<Button>Default Button</Button>
```

### Button Variant

Use the `variant` prop to change the visual style of the Button. You can set the value to `solid`, `outline`. Use `outline` to remove all background images and colors on any button.

```jsx
<div>
  <Button variant="solid">Button</Button>
  <Button variant="outline">Button</Button>
</div>
```

### Button Color

Use the `variantColor` prop to change the color scheme of the Button. `variantColor` can be any color key with key `50`, `60` and `70` that exist in the them.color.

```jsx
<div>
  <Button variantColor="red">Emphasis Button</Button>
  <Button variantColor="blue">Primary Button</Button>
  <Button variantColor="gray">Default Button</Button>
</div>
```

### Button Sizes

Use the `size` prop to change the size of the button. You can set the value to `sm`, `md`, or `lg`.

```jsx
<div>
  <Button size="sm">Button</Button>
  <Button size="md">Button</Button>
  <Button size="lg">Button</Button>
</div>
```

## Accessibility

- Button has `role` button.
- When Button has focus, <kbd>Space</kbd> and <kbd>Enter</kbd> activates it.

## Composition

Button composes [PseudoBox](./pseudobox) and all props you pass (variant, variantColor, etc.) are converted to style props. This means you can override any style of the Button via props.

```jsx
<Button
  variant="outline"
  height="48px"
  width="200px"
  __before={{
    border: 'none'
  }}
>
  Ghost Button
</Button>
```
### Button with Icon

You can add icons to the Button components.

```jsx
<div>
  <div>
    <Button variantColor="blue" borderRadius="circle">+</Button>
    <Button variantColor="blue">+</Button>
    <Button variantColor="blue" borderRadius="32px">+ Add Account</Button>
    <Button variantColor="blue">+ Add Account</Button>
  </div>
  <div>
    <Button variant="outline" borderRadius="circle">+</Button>
    <Button variant="outline">+</Button>
    <Button variant="outline" borderRadius="32px">+ Add Account</Button>
    <Button variant="outline">+ Add Account</Button>
  </div>
</div>
```

## Custom Button

In event you need to make your own custom button, you can leverage the
`PseudoBox` component. It provides the `hover`, `focus`, `active` and `disabled` style props to style the button.

```jsx
// Button from facebook.com
<PseudoBox
  as="button"
  height="24px"
  lineHeight="1.2"
  transition="all 0.2s cubic-bezier(.08,.52,.52,1)"
  border="1px solid"
  px="8px"
  rounded="2px"
  fontSize="14px"
  fontWeight="semibold"
  bg="#f5f6f7"
  borderColor="#ccd0d5"
  color="#4b4f56"
  _hover={{ bg: "#ebedf0" }}
  _active={{
    bg: "#dddfe2",
    transform: "scale(0.98)",
    borderColor: "#bec3c9",
  }}
  _focus={{
    boxShadow:
      "0 0 1px 2px rgba(88, 144, 255, .75), 0 1px 1px rgba(0, 0, 0, .15)",
  }}
>
  Join Group
</PseudoBox>
```

## Props

The Button composes the `PseudoBox` component so you can pass props for `PseudoBox`. These are props related to the Button component.

| Name               | Type                 | Default       | Description                                                            |
| ------------------ | -------------------- | ------------- | ---------------------------------------------------------------------- |
| `disabled`         | boolean              | `false`       | If `true`, the button will be disabled.                                |
| `size`             | `sm`, `md`, `lg`     | `md`          | The size of the button.                                                |
| `variant`          | `solid`, `outline`   | `solid`       | The variant of the button style to use.                                |
| `variantColor`     | string               |               | The color of the button. Use a color key passed in `theme.colors`.     |
